# Income-Prophet

Income forecasting tool using time-series modelling and growth projection

Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

## Overview

Predict future income streams using statistical time-series models and machine learning. Input historical earnings data and receive probabilistic forecasts with confidence intervals for financial planning.

## Features

- Time-series decomposition and trend analysis
- ARIMA and exponential smoothing models
- Growth rate projection and scenario analysis
- Confidence interval estimation
- Multi-period forecasting with visualization

## Tech Stack

Python · pandas · time-series forecasting

## Setup

```
pip install -r requirements.txt
python main.py
```
